# 📖 [Awesome Books] <a name="about-project"></a>


**[Awesome Books]** is a single page application (SPA) that allow users to store a book with title and author name and add to their books list on the page.

## 🛠 Built With
 <a name="built-with">
  - HTML
  - CSS
  - JavaScript
 </a>
